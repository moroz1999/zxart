# zx-inset

Padding layout container with restricted spacing values.

`ng-zxart/src/app/shared/ui/zx-inset/`

## Props

| Prop | Values | Default |
|---|---|---|
| `padding` | `none` \| `xs` \| `sm` \| `md` \| `lg` \| `xl` | `md` |
| `side` | `all` \| `block` \| `inline` \| `top` \| `right` \| `bottom` \| `left` | `all` |
