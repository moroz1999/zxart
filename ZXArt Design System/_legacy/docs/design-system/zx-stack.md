# zx-stack

Vertical flexbox layout container.

`ng-zxart/src/app/shared/ui/zx-stack/`

## Props

| Prop | Values | Default |
|---|---|---|
| `gap` / `spacing` | `none` \| `xs` \| `sm` \| `md` \| `lg` \| `xl` \| `xxl` | `md` |
| `align` | `center` \| `start` \| `end` \| `stretch` | — |

Use `zx-inline` for horizontal layout.
