# zx-icon-popover

`zx-icon-popover` renders an icon-only `zx-button` trigger and shows projected content in a CDK connected overlay.

Use it for compact contextual actions that must stay close to an item title or control. The popover content is arranged as a vertical `zx-stack`; pass reusable controls or buttons as projected content.

The component is generic. Feature-specific labels, action configuration, and permissions stay in the feature or domain layer.
