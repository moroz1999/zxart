# zx-filter-picker

Compact summary control with a popover list of checkboxes and optional search.

`ng-zxart/src/app/shared/ui/zx-filter-picker/`

## Props

| Prop | Type |
|---|---|
| `label` | string |
| `items` | `{id, label}[]` |
| `selectedIds` | any[] |
| `placeholder` | string |
| `searchEnabled` | boolean |
| `multi` | boolean |

## Events

| Event | Payload |
|---|---|
| `selectedIdsChange` | updated selection array |
