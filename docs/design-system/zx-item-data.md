# zx-item-data

Grouped key-value information block.

`ng-zxart/src/app/shared/ui/zx-item-data/`

## Components

- `zx-item-data` wraps grouped data in a flat panel.
- `zx-item-data-item` renders one label/value pair as a grid row.

## Rules

Use `zx-item-data` for compact entity facts where labels and values should scan as a group.

Labels are visually muted; values keep normal body styling. Values may contain links or other inline design-system components.
