# zx-table

Table wrapper inside an elevated panel with `contentBleed`. Table rows go edge-to-edge while the title retains standard panel padding.

`ng-zxart/src/app/shared/ui/zx-table/`

## Props

| Prop | Values |
|---|---|
| `title` | string (optional heading above the table) |
| `titleLevel` | `h2` \| `h3` |
