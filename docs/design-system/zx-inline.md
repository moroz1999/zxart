# zx-inline

Horizontal flexbox layout container.

`ng-zxart/src/app/shared/ui/zx-inline/`

## Props

| Prop | Values | Default |
|---|---|---|
| `gap` / `spacing` | `none` \| `xs` \| `sm` \| `md` \| `lg` \| `xl` \| `xxl` | `md` |
| `align` | `center` \| `start` \| `end` \| `stretch` | — |
| `justify` | `start` \| `center` \| `end` \| `between` | — |
| `wrap` | `boolean` | `false` |
