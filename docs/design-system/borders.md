# Borders

Standard border tokens for inputs and containers:

| Token | Value | Use when |
|---|---|---|
| `--border-width-thin` | 1px | Default border width |
| `--border-width-thick` | 2px | Emphasized borders |
| `--border-secondary` | Gray | Default containers, `flat` panels |
| `--border-primary` | Blue | Active or selected states |
| `--border-focused` | High-contrast | Focused inputs and interactive elements |
