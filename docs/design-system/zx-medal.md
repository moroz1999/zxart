# zx-medal

`zx-medal` displays a compact circular numbered badge.

Use `zx-medal` only as a UI primitive. Domain logic, such as choosing the variant from a party place, belongs in shared-lib components.

Variants:
- `gold` - first place.
- `silver` - second place.
- `bronze` - third place.
- `outlined` - transparent background, standard border, and standard text color.
