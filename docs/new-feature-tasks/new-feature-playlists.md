Нужно на ангуляре воспроизвести функционал добавления в favorites.
Оно выглядит как сердечко, нажав которое, с сервера скачиваются списки favourites пользователя. Появляется Popover:
- вертикально показан список названий списков избранного.
- по наведению мышки на каждый список справа от названия появляется подсказка "добавить в список"
- по клику на список отсылается запрос на сервер и работа добавляется в список. справа от названия списка показывается зеленая галочка.
- повторный клик убирает работу из списка.
- внизу поповера есть мини-форма создания нового. input с плейсхолдеров "Имя нового списка" и справа кнопка "Добавить". На одной строчке.
- после добавления идет запрос на сервер, и список избьранного добавляется в список в поповере.
- добавить можно музыку, графику, prod, release
