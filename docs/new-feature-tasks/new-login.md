В шапке есть форма логина, делаем её в ангуляр.
1. По ответу currentuser запроса понимаем, логинен или нет.
2. надо посмотреть имеющуюся и сделать форму логина. Юзер, пасс, "запомнить" чекбокс. 
3. Доделать currentuser контроллер для логина. 
4. Глянуть, как логин сделан в легаси CMS модуле. Из легаси всю логику логина вытащить в новый сервис.
5. Использовать этот сервис для currentuser. Старый модуль логина не трогать - он в старой админке использован и пока ещё нужен.
6. Форму регистрации оставить пока старой. Ссылки на форму регистрации передавать в том же ajax+сервисе, что и ссылки каталога. 
7. Ввести директиву zx-form, которая будет регулировать отступы элементов форм.
